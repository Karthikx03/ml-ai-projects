## 🚀 Projects Overview

### 🛡️ PhishGuard AI
> AI-powered phishing URL detection using ensemble ML classifiers

- **Accuracy**: ~95%+ on test datasets
- **Techniques**: Feature engineering, Random Forest, XGBoost
- **Stack**: Python, Scikit-Learn, Flask, HTML/CSS
- 🔗 [View Repo](https://github.com/Karthikx03/phishguard-ai)

### 📊 Supervised Learning Collection
> Classic ML algorithms implemented from scratch and with sklearn

- Linear/Logistic Regression, Decision Trees, SVM
- Each with detailed Jupyter notebooks + visualizations
- **Stack**: Python, NumPy, Pandas, Matplotlib, Scikit-Learn

### 🧠 Deep Learning Experiments
> Neural network architectures for vision and language tasks

- CNN for image classification (CIFAR-10, custom datasets)
- RNN/LSTM for sequence modeling
- Transformer architecture exploration
- **Stack**: PyTorch, TensorFlow/Keras

### 💬 NLP Projects
> Natural Language Processing models and experiments

- Sentiment analysis (Twitter dataset)
- Text classification with BERT fine-tuning
- Named Entity Recognition
- **Stack**: Python, NLTK, Transformers (HuggingFace), spaCy

## 🛠️ Setup & Usage

```bash
# Clone the repository
git clone https://github.com/Karthikx03/ml-ai-projects.git
cd ml-ai-projects

# Create virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Launch Jupyter
jupyter notebook
```
